daemon: 컨테이너 오케스트레이션. cli로부터 명령 받아 컨테이너를 돌린다. 생명 주기 관리. cli로부터 docker run 등의 명령어를 받아 처리.

이미지: 런타임, 라이브러리, 코드 등의 돌리는데 필요한 명세들이 들어간 일종의 명세, 빌드 결과물

[[개발/컨테이너\|컨테이너]]: 이미지로 돌리는 하나의 인스턴스 프로세스

docker compose: [[개발/yaml\|yaml]] 파일에 적힌 명세를 읽고, 거기에 따라 여러 컨테이너들을 한번에 띄우는 명령어
